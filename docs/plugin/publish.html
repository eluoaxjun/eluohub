Publish v1.0.0

디자인 시안을 프로덕션 코드로 변환합니다. 시맨틱 HTML5 마크업, CSS Custom Properties 기반 스타일링, 바닐라 JS 인터랙션을 3단계로 구현합니다.

포함 자산

Skills (4)

스킬명설명
publish퍼블리싱 전체 파이프라인 — Markup → Style → Interaction 순차 실행
publish-markup시맨틱 HTML 마크업 — BEM 네이밍, SEO, WCAG 2.1 AA 접근성
publish-styleCSS 스타일링 — 디자인 토큰 → CSS Custom Properties, 모바일 퍼스트 반응형
publish-interactionJS 인터랙션 — FN 명세 기반 바닐라 JS, ARIA 상태 관리, 키보드 내비게이션

Commands (1)

커맨드설명
/publish:publish퍼블리싱 전체 (Markup → Style → Interaction)

Agents (2)

에이전트설명
publish-orchestrator퍼블리싱 오케스트레이터 — 이미지 자동소싱, Playwright 3뷰포트 렌더링 검증, 버전관리
publish-reviewer퍼블리싱 검수자 — 디자인 QA 100점 + 이미지 검증 + 3단계 QA

Planning v1.0.0

고객질의서(QST)부터 요구사항(REQ), 기능정의(FN), 정보구조(IA), 작업분해(WBS), 화면설계서(SB), 대시보드까지 기획 전체 산출물을 자동 생성합니다.

포함 자산

Skills (8)

스킬명설명
plan기획 전체 파이프라인 (QST → REQ → FN → IA → WBS)
plan-qst고객질의서 — 비즈니스 목표, KPI, 팀 구성, 예산/일정 파악
plan-req요구사항정의서 — 기능/비기능 요구사항, AC, NFR 측정기준
plan-fn기능정의서 — 상세 기능 명세, 복잡도별 분기, 의존관계 맵
plan-ia정보구조설계 — 사이트맵, 페이지 인벤토리, 네비게이션, URL 설계
plan-wbs작업분해구조 — 일정 분해, 간트 차트, 크리티컬 패스
plan-SB화면설계서 — 와이어프레임 기반 화면 명세
plan-dashboard대시보드 — 전체 산출물 현황, KPI, 이슈, 마일스톤 종합

Claude Code 설치
이미 설치되어 있다면 이 단계를 건너뛰세요.
설치 여부 확인

터미널(또는 명령 프롬프트)에서 아래 명령어를 실행합니다.

claude --version

버전 번호가 출력되면 Step 1로 이동하세요. 'claude'을(를) 찾을 수 없습니다 등 에러가 나오면 아래 설치를 진행합니다.

Windows
PowerShell (관리자) 
irm https://claude.ai/install.ps1 | iex

Git for Windows가 필요합니다. 미설치 시: winget install --id Git.Git -e

macOS
Terminal 
curl -fsSL https://claude.ai/install.sh | bash

Node.js 불필요. 설치 후 터미널을 재시작하세요.

설치 후 최초 인증
claude

처음 실행하면 브라우저가 열리며 Anthropic 계정 로그인을 요청합니다. 로그인 완료 후 터미널로 돌아오면 인증 완료입니다.

1
스킬 폴더 다운로드
필요한 스킬만 골라서 다운로드합니다. 각 폴더는 자기완결형이므로 하나만 받아도 바로 사용할 수 있습니다.
예시
스킬폴더명설명
고객질의서plan-qst/고객 요구사항 수집
요구사항정의서plan-req/기능·비기능 요구사항 정의
기능정의서plan-fn/입력-처리-출력 상세 명세

전체 스킬 목록은 스킬 카탈로그를 참고하세요.

2
스킬 폴더에서 Claude Code 실행
다운로드한 폴더를 열고 Claude Code를 실행합니다.
예시
폴더 구조 
plan-req/                    ← 이 폴더에서 Claude Code 실행
├── .claude/                 ← 스킬 설정 (자동 인식, 수정 불필요)
├── README.md                ← 사용 설명
├── input/                   ← 참고 자료 넣는 곳
└── output/                  ← 산출물이 여기에 생성됩니다
참고 자료 넣기

input/ 폴더에 아래 자료를 넣으면 산출물 품질이 향상됩니다.

  • 제안서, RFP
  • 디자인 시안 (이미지, PDF)
  • 현행 사이트 분석 자료
  • 벤치마킹 레퍼런스
PROJECT.md 직접 작성하기 (선택)

보통은 스킬이 자동 생성하지만, 미리 작성해 두면 질문 없이 바로 진행됩니다.

PROJECT.md 
# 프로젝트명

## 개요
- 고객사: OOO
- 프로젝트 유형: 신규 구축 / 리뉴얼 / 운영
- 오픈 예정일: YYYY-MM-DD

## 주요 요구사항
- (고객이 전달한 핵심 내용을 기술)
3
동작 테스트
자연어로 요청하면 산출물이 자동 생성됩니다.
예시
💬 "요구사항 정리해줘"

output/ 폴더에 산출물 파일이 생성되면 설치 완료입니다.

MCP 서버

대부분의 스킬은 MCP 서버가 필요 없습니다.
일부 스킬(디자인 벤치마킹, QA 성능 테스트 등)은 Playwright 등 MCP 서버가 필요할 수 있으며, 해당 스킬의 README에 안내가 포함됩니다.

업데이트

새 버전을 배포받으면 스킬 폴더의 .claude/만 덮어쓰기합니다.
input/, output/ 등 작업 데이터는 영향 없습니다.

제거

스킬 폴더를 삭제합니다. 산출물을 보관하려면 output/만 별도로 백업하세요.

문제 해결
증상원인해결
자연어 요청에 반응 안 함 Claude Code 재시작 필요 Ctrl+C로 종료 후 재실행
산출물이 생성되지 않음 스킬 폴더 밖에서 실행 스킬 폴더 안에서 Claude Code 실행
스킬이 동작 안 함 .claude/ 경로 오류 .claude/skills/ 하위에 스킬 폴더 존재 확인

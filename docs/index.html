설치 가이드

        
3단계로 시작하세요

      


      

        

          

            
1

            

              
마켓플레이스 등록

              
Claude Code에서 Eluo Hub 마켓플레이스를 등록합니다.

              

                

                  Claude Code
                  
                

                
/plugin marketplace add eluoaxjun/eluohub

              

            

          


          

            
2

            

              
플러그인 설치

              
core는 필수, 나머지는 필요한 것만 선택 설치합니다.

              

                

                  Claude Code
                  
                

                
/plugin install core@eluo-hub
/plugin install planning@eluo-hub

              

            

          


          

            
3

            

              
프로젝트 설정 (선택)

              
팀원 자동 안내를 위해 프로젝트 settings.json에 추가합니다. 위 설정 생성기에서 복사할 수 있습니다.

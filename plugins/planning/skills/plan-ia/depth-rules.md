# Depth 계층 및 네비게이션 규칙

## Depth별 역할

### 1depth — 메인 카테고리
- GNB에 직접 노출
- **최대 7개** 권장 (밀러의 법칙)
- 사이트 핵심 영역을 대표
- ID 범위: IA-P100, IA-P200, ... IA-P700

### 2depth — 서브 카테고리
- LNB에 노출
- 1depth 하위 세분화
- 하나의 1depth에 **2~8개** 권장
- ID 범위: IA-P101~P199, IA-P201~P299, ...

### 3depth — 상세 페이지
- 실질적 콘텐츠 페이지
- 동적 콘텐츠 (게시글 상세, 상품 상세) 포함
- 브레드크럼 필수 적용
- ID 범위: IA-P110~P199, IA-P210~P299, ...

### 4depth — 특수 페이지
- 모달, 팝업, 레이어
- **최소화** 원칙 (가능하면 3depth로 처리)

## 업종별 표준 구조

### 기업 홈페이지
```
홈 / 회사소개 / 사업영역 / 뉴스·공지 / 고객지원
```

### 쇼핑몰
```
홈 / 상품(카테고리별) / 브랜드 / 커뮤니티 / 고객센터
+ 주문결제 플로우 / 마이페이지
```

### 교육 사이트
```
홈 / 강좌(카테고리별) / 강사 / 수강후기 / 고객센터
+ 수강신청 플로우 / 마이클래스
```

### 공공기관
```
홈 / 기관소개 / 정보공개 / 정책·사업 / 참여소통 / 알림마당
```

## 공통 필수 페이지
- 홈 (IA-P001)
- 통합검색 (IA-P9XX)
- 개인정보처리방침 (법적 필수)
- 이용약관
- 404 에러 페이지
- 500 에러 페이지

## 네비게이션 패턴

### GNB 패턴
| 패턴 | 적합 | 특징 | 모바일 |
|------|------|------|--------|
| 기본 수평 | 5개 이하 1depth | 심플, 빠른 인지 | 햄버거 전환 |
| 메가메뉴 | 6~7개 1depth + 다수 2depth | 호버 시 전체 노출 | 아코디언 전환 |
| 탭 네비게이션 | 3~5개 동등한 섹션 | 콘텐츠 전환, SPA | 스크롤 탭 |

### LNB 패턴
| 패턴 | 적합 | 특징 | 모바일 |
|------|------|------|--------|
| 좌측 사이드바 | 5개 이상 2depth | 상시 노출, 아코디언 | 상단 드롭다운 |
| 상단 서브 탭 | 3~5개 2depth | 수평 배치 | 스크롤 탭 |
| 필터 네비게이션 | 상품/검색 결과 | 다중 필터 조합 | 바텀시트 필터 |

### Footer 패턴
| 패턴 | 적합 | 구성 |
|------|------|------|
| 멀티 컬럼 | 대규모 사이트 | 3~4열 링크 + 법적고지 + SNS |
| 미니멀 | 소규모 사이트 | 1줄 링크 + 저작권 |

### 브레드크럼 패턴
| 패턴 | 형식 | 적합 |
|------|------|------|
| 경로 기반 | 홈 > 카테고리 > 서브 > 현재 | 계층 구조 명확한 사이트 |
| 속성 기반 | 홈 > 속성값 > 속성값 > 현재 | 필터 기반 탐색 (커머스) |

### 유틸리티 네비게이션 상태별 변경
- 비로그인: 로그인 | 회원가입
- 로그인: {이름}님 | 마이페이지 | 로그아웃
- 관리자: + 관리자 페이지

### 검색 패턴
- 아이콘 + 확장형 (클릭 시 입력창 확장)
- 상시 노출형 (입력창 항상 표시)
- 오버레이형 (전체화면 검색)

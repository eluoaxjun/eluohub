# FN 체크리스트

## 필수 항목
- [ ] REQ 기반 ID 매핑 확인 `[연계 모드 전용]`
- [ ] 모든 FN에 고유 ID (FN-###)

## 기능 카드 헤더
- [ ] 상태 (개발 대기/진행중/완료)
- [ ] 복잡도 (서술형 + 등급)
- [ ] 우선순위 (Must/Should/Could)
- [ ] 연관 REQ (FR-### 목록) `[연계 모드 전용]`
- [ ] 담당 (역할/팀)

## 상세 명세 (복잡도별)
- [ ] 높음: 4-탭 (입력/처리/출력/에러) 완비
- [ ] 중간: 처리 프로세스 + 에러 처리
- [ ] 낮음: 처리 프로세스 서술

## 검증 기준 (복잡도 중간 이상)
- [ ] 정상 시나리오 정의
- [ ] 예외 시나리오 정의
- [ ] 에러 시나리오 정의

## 비즈니스 가치 (높음 또는 Must만)
- [ ] 비즈니스 목표 연결
- [ ] 대상 사용자
- [ ] 예상 사용 빈도
- [ ] 비즈니스 영향 체인

## 의존관계
- [ ] FN 간 선후행 관계 명시
- [ ] Gate 3에서 의존관계 맵 출력
- [ ] 독립 FN은 "독립" 표기

## 추적성
- [ ] FR-### -> FN-### 매핑 `[연계 모드 전용]`
- [ ] 추적성 매트릭스 갱신 `[연계 모드 전용]`
- [ ] [미확인] 0건 확인

## 운영 모드
- [ ] 모든 변경 FN에 AS-IS 현행 기술 (화면/처리/제약)
- [ ] 모든 변경 FN에 TO-BE 변경 기술 (대비 형식)
- [ ] 모든 변경 FN에 변경 이유 명시
- [ ] 검증 기준이 AS-IS→TO-BE 변경점 중심
- [ ] 영향 범위 4유형 분류 (직접/연쇄/참조/무관)
- [ ] 롤백 방안 3필드 (트리거/방법/영향) 필수

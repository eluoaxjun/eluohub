# QST 체크리스트

## 필수 항목
- [ ] 프로젝트명/고객사/담당자 기본정보 확인
- [ ] 프로젝트 유형 (구축/운영) 판별
- [ ] 복잡도 레벨 결정 (Level 1/2)

## 구축 QST (4단계)
- [ ] 1단계: 비즈니스 목표 + KPI 질의
- [ ] 2단계: 모니터링 + 대시보드 요구 질의
- [ ] 3단계: 팀 구성 + 역할 질의
- [ ] 4단계: 예산 + 일정 + 제약 질의
- [ ] 업종별 특화 질의 추가 (해당 시)

## 운영 QST
- [ ] 현황 파악 (문제/불편, 빈도, 영향 범위)
- [ ] 운영 효율성 지표 (처리 시간, 오류율)
- [ ] 즉시/단기/중기 분류

## 품질
- [ ] 모든 질의가 30초 답변 가능한 선택형
- [ ] KPI가 구체적 수치 빈칸 포함
- [ ] [미확인] 태그 적절히 부여
- [ ] 질의 ID (Q-###) 부여

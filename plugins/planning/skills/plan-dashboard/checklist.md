# 대시보드 품질 체크리스트

## 산출물 스캔
- [ ] output/planning/ 디렉토리 스캔 완료
- [ ] 존재하는 산출물 목록 확인 (QST/REQ/FN/IA/WBS)
- [ ] 각 산출물 버전 확인

## 지표 추출
- [ ] QST: 총 질의 수, 확인/미확인 비율 추출
- [ ] REQ: FR 수 (High/Med/Low), NFR 수, BG 수 추출
- [ ] FN: 총 기능 수, 복잡도 분포, 의존관계 수 추출
- [ ] IA: 총 페이지 수, Depth 분포, GNB 수 추출 (해당 시)
- [ ] WBS: 총 작업 수, 총 기간, 크리티컬 패스 추출 (해당 시)

## 이슈 정리
- [ ] QST [미확인] 잔여 건수 집계
- [ ] FN 의존관계 병목 식별 (팬인 3개 이상)
- [ ] WBS 리스크 반영 (해당 시)

## 대시보드 형식
- [ ] 프로젝트 개요 섹션 완비
- [ ] 산출물 현황 섹션 (완료/진행/대기 표시)
- [ ] KPI 섹션 (일정/품질/범위)
- [ ] 이슈 섹션 (심각도 + 내용 + 대응)
- [ ] 마일스톤 섹션
- [ ] 전체 진행률 계산 정확성

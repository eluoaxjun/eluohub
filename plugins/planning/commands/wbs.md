# 작업분해구조 (WBS) Generator

프로젝트 작업분해구조를 생성합니다.

## WBS 구조

### 1. 프로젝트 일정 대시보드
```
📅 프로젝트 일정 현황
├── 시작일: ____년 __월 __일
├── 종료일: ____년 __월 __일
├── 총 기간: __주
├── 진행률: __%
└── 상태: 🟢정상 / 🟡주의 / 🔴지연
```

### 2. 단계별 작업 분해
```
📋 Phase 1: 기획 및 분석 (1-2주차)
├── 1.1 요구사항 분석 (QST/REQ)
│   ├── 고객 미팅: 2일
│   ├── 질의서 작성: 1일
│   └── 요구사항 정리: 2일
├── 1.2 기능 정의 (FN)
│   ├── 기능 목록 작성: 2일
│   └── 상세 정의: 3일
└── 마일스톤: 기획 완료 ✓

📋 Phase 2: 설계 (3-4주차)
├── 2.1 정보구조 설계 (IA)
├── 2.2 화면 설계 (UI)
└── 마일스톤: 설계 완료 ✓

📋 Phase 3: 개발 (5-8주차)
├── 3.1 프론트엔드 개발
├── 3.2 백엔드 개발
├── 3.3 DB 구축
└── 마일스톤: 개발 완료 ✓

📋 Phase 4: 테스트 및 런칭 (9-10주차)
├── 4.1 통합 테스트 (QA)
├── 4.2 사용자 테스트
├── 4.3 런칭 준비
└── 마일스톤: 프로젝트 완료 ✓
```

### 3. 간트 차트 연동
- 작업별 시작/종료일
- 의존 관계 표시
- 크리티컬 패스 강조

### 4. 리소스 배정
- 역할별 담당자
- 작업 부하 분산
- 병목 지점 식별

## 출력 형식
- HTML 간트 차트
- 인터랙티브 일정표
- 실시간 진행률 추적

$ARGUMENTS

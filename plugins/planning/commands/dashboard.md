# 프로젝트 대시보드 Generator

프로젝트 현황을 한눈에 파악하는 대시보드를 생성합니다.

## 대시보드 구성

### 1. 프로젝트 개요
```
┌─────────────────────────────────────┐
│ 📊 [프로젝트명] 대시보드              │
├─────────────────────────────────────┤
│ 📅 기간: ____.__.__~ ____.__.__     │
│ 👥 팀: [팀 구성원]                   │
│ 💰 예산: ___,___,___ 원              │
│ 📈 진행률: ██████████░░ 75%          │
└─────────────────────────────────────┘
```

### 2. KPI 현황
```
┌──────────┬──────────┬──────────┐
│  📈 일정  │  💰 예산  │  ✅ 품질  │
├──────────┼──────────┼──────────┤
│ 목표: 10주│ 목표:50M │ 목표:<5% │
│ 현재: 7주 │ 사용:35M │ 현재: 3% │
│ 🟢 정상   │ 🟢 정상   │ 🟢 양호   │
└──────────┴──────────┴──────────┘
```

### 3. 산출물 진행 현황
```
📋 산출물 체크리스트
├── ✅ QST (고객질의서) - 완료
├── ✅ REQ (요구사항정의서) - 완료
├── 🔄 FN (기능정의서) - 진행중 (80%)
├── ⏳ IA (정보구조설계) - 대기
├── ⏳ UI (화면설계서) - 대기
├── ⏳ WBS (작업분해구조) - 대기
└── ⏳ QA (테스트계획서) - 대기
```

### 4. 이슈 현황
```
⚠️ 이슈 현황
├── 🔴 Critical: 0건
├── 🟡 Major: 2건
├── 🟢 Minor: 5건
└── 총 미해결: 7건
```

### 5. 마일스톤 타임라인
```
──●────────●────────●────────○────────○──
  기획완료   설계완료   개발50%   테스트    런칭
  (완료)    (완료)    (진행중)  (예정)    (예정)
```

### 6. 팀 현황
```
👥 팀 업무 현황
├── PM: 전체 관리, 고객 소통
├── 기획: FN 작성 중
├── 디자인: UI 준비 중
├── 개발: 환경 구축 완료
└── QA: 테스트 계획 수립 중
```

## 출력 형식
- HTML 인터랙티브 대시보드
- 실시간 데이터 연동
- 인쇄 가능 버전 제공

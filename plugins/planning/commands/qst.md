# 고객질의서 (QST) Generator

고객 요구사항 파악을 위한 질의서를 생성합니다.

## 프로젝트 유형 판단
- **구축**: 신규 제작, 전면 리뉴얼, 대규모 기능 추가
- **운영**: 기존 수정, 기능 개선, 버그 수정

## 복잡도 레벨
- **Level 1 (단순)**: 15분 템플릿, 핵심 질의 3개
- **Level 2 (복잡)**: 30분 템플릿, 상세 질의 5개+

## 질의서 템플릿

### 기본 정보
```
- 프로젝트명: [입력]
- 고객사: [입력]
- 담당자: [입력]
- 접수일: [자동]
- 복잡도: Level [1/2]
```

### 핵심 질의 구조 (30초 답변 가능)
```
📊 KPI 설정:
A) 매출 증대 (목표: ___% 증가)
B) 고객 확대 (목표: ___배 증가)
C) 효율성 향상 (목표: ___% 단축)
D) 고객만족도 (목표: ___점/5점)

⏱️ 일정:
- 런칭 목표일: ____년 __월 __일
- 주요 마일스톤: [선택형 제공]

💰 예산:
A) 1천만원 미만
B) 1천만원 ~ 3천만원
C) 3천만원 ~ 5천만원
D) 5천만원 이상
```

## 출력 형식
- HTML 탭 기반 인터랙티브 문서
- 프로젝트 대시보드 연동 가능

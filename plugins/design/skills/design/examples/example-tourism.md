## 브랜드 시트 발췌

### 섹션 0: 디자인 방향
톤: **세련됨(Refined), 신뢰감(Trustworthy), 활력(Vibrant), 접근성(Accessible)** | 타겟: 외국인 FIT + 내국인 감성 관광객

### 섹션 2: 디자인 토큰

| 토큰명 | 값 | 용도 |
|--------|-----|------|
| --color-primary | #1B3A5C | CTA, 강조, GNB (네이비 — 도시적 세련됨) |
| --color-secondary | #E85D3A | 포인트 CTA, 뱃지 (코럴 — 관광 활력) |
| --color-neutral-900 | #1A1A1A | 본문 텍스트 |
| --color-neutral-500 | #666666 | 보조 텍스트 |
| --color-bg | #FAFAFA | 기본 배경 |
| --color-success | #2E8B57 | 진행중 뱃지, 성공 상태 |
| --font-heading | Pretendard | 제목 (KO) |
| --font-body | Inter | 본문 (EN, KO 혼용) |
| --text-h1 | 40px(D) → 32px(M) | 히어로 타이틀 |
| --text-h2 | 32px(D) → 26px(M) | 섹션 타이틀 |
| --text-body | 16px | 본문 고정 |
| --space-section | 64px(D) → 48px(T) → 32px(M) | 섹션 간 여백 |
| --space-card-gap | 24px(D) → 16px(M) | 카드 간격 |
| --radius-card | 12px | 카드 모서리 |
| --radius-button | 8px | 버튼 모서리 |

### 섹션 3: 컴포넌트 규칙

| 컴포넌트 | 규칙 |
|---------|------|
| 버튼(Primary) | bg #1B3A5C / text white / radius 8px / 48px 높이 |
| 버튼(Secondary) | border 1px #1B3A5C / text #1B3A5C / bg transparent |
| 카드(행사) | 이미지 3:2 / radius 12px / hover translateY(-4px) + shadow 강화 |
| 카드(코스) | 이미지 16:9 / radius 12px / 하단 소요시간+스팟수 |
| 카드(여행지) | 이미지 1:1 / radius 12px / 하단 카테고리 태그 |
| 필터 칩 | 기본 bg #f3f4f6 radius 20px / 선택 bg #1B3A5C text white |
| 아코디언 | chevron 180도 회전 / slide-down 300ms ease |
| 상태 뱃지 | 좌상단 absolute, pill shape, 진행중=#2E8B57 종료=#999 |

### 섹션 5: Do's & Don'ts
- Do: 이미지에 충분한 여백 확보 (이미지 80% + 여백 20%)
- Do: CTA 버튼은 페이지당 1~2개로 제한
- Do: 카드 호버 시 부드러운 elevation 변화 (300ms ease)
- Do: 컬러는 Primary+Secondary 2색 위주, Accent는 상태 표시만
- Don't: 히어로에 3줄 이상 텍스트
- Don't: 한 화면에 3가지 이상 카드 스타일 혼용
- Don't: 그라데이션 배경 (공공 사이트 신뢰감 저하)
- Don't: 애니메이션 0.5초 초과 (접근성 prefers-reduced-motion)

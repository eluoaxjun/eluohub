## 브랜드 시트 발췌

### 섹션 0: 디자인 방향
톤: **미니멀, 세련, 여성적, 프리미엄** | 타겟: 2030 여성

### 섹션 2: 디자인 토큰

| 토큰명 | 값 | 용도 |
|--------|-----|------|
| --color-primary | #1A1A1A | CTA, 강조 텍스트 |
| --color-accent | #C8A97E | 포인트 (골드) |
| --color-bg | #FAFAFA | 배경 |
| --color-bg-alt | #F0EDE8 | 섹션 구분 |
| --font-heading | Pretendard | 제목 |
| --font-body | Pretendard | 본문 |
| --text-h1 | 2.5rem | 페이지 타이틀 |
| --text-h2 | 1.75rem | 섹션 타이틀 |
| --text-body | 1rem | 본문 |
| --space-section | 80px | 섹션 간 여백 |
| --space-card-gap | 20px | 카드 간격 |

### 섹션 3: 컴포넌트 규칙

| 컴포넌트 | 규칙 |
|---------|------|
| 버튼(Primary) | #1A1A1A / #fff / radius 0 / 48px 높이 |
| 버튼(Secondary) | transparent / #1A1A1A / 1px solid / radius 0 |
| 카드(상품) | 이미지 3:4 비율 / 하단 텍스트 좌정렬 / 호버 시 이미지 scale(1.03) |
| 입력필드 | 하단 보더만 / 48px 높이 / 포커스 시 #1A1A1A |

### 섹션 5: Do's & Don'ts
- Do: 여백을 충분히 — 고급감의 핵심
- Do: 상품 이미지 비율 통일 (3:4) — 그리드 정렬
- Don't: 팝업/배너 남용 — 브랜드 이미지 저해
- Don't: 3가지 이상 폰트 사용 — 산만해짐

## UI 발췌: 상품 목록 (UI-002)

| 항목 | 내용 |
|------|------|
| IA 페이지 | IA-P102 (카테고리 목록) |
| FN 매핑 | FN-003 (상품 목록 조회), FN-004 (필터/정렬) |
| Layout | Layout-002 참조 |

**UI 요소**

| 영역 | 요소 | 상세 |
|------|------|------|
| 상단 | 카테고리 브레드크럼 | 홈 > 여성 > 아우터 |
| 상단 | 필터 바 | 가격/색상/사이즈/브랜드 (드롭다운) |
| 상단 | 정렬 | 신상품순/인기순/가격순 (Select) |
| 본문 | 상품 카드 그리드 | 4열 / 이미지(3:4) + 브랜드 + 상품명 + 가격 |
| 하단 | 페이지네이션 | 숫자형 (1 2 3 ... 10) |

**상태별 화면**

| 상태 | 표시 |
|------|------|
| Default | 카드 그리드 + 필터 |
| Loading | 카드 스켈레톤 (4x3) |
| Empty | "조건에 맞는 상품이 없습니다" + 필터 초기화 버튼 |
| Error | "상품을 불러올 수 없습니다" + 새로고침 버튼 |

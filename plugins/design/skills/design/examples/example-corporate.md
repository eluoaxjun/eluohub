## 브랜드 시트 발췌

### 섹션 0: 디자인 방향
톤: **신뢰, 전문, 깔끔, 혁신** | 타겟: B2B 의사결정자

### 섹션 2: 디자인 토큰

| 토큰명 | 값 | 용도 |
|--------|-----|------|
| --color-primary | #2563EB | CTA, 링크, 강조 |
| --color-secondary | #1E40AF | 호버, 진한 강조 |
| --color-text | #111827 | 본문 텍스트 |
| --color-text-sub | #6B7280 | 보조 텍스트 |
| --color-bg | #FFFFFF | 배경 |
| --color-bg-alt | #F9FAFB | 섹션 구분 |
| --font-heading | Pretendard | 제목 |
| --font-body | Pretendard | 본문 |
| --text-h1 | 3rem | Hero 타이틀 |
| --text-h2 | 2rem | 섹션 타이틀 |
| --text-body | 1rem | 본문 |
| --text-caption | 0.875rem | 캡션 |
| --space-section | 120px | 섹션 간 여백 (넓게) |
| --space-card-gap | 24px | 카드 간격 |

### 섹션 3: 컴포넌트 규칙

| 컴포넌트 | 규칙 |
|---------|------|
| 버튼(Primary) | #2563EB / #fff / radius 8px / 48px 높이 / 그림자 |
| 버튼(Secondary) | #fff / #2563EB / 1px solid / radius 8px |
| 카드(서비스) | #fff / radius 12px / shadow-md / 패딩 32px / 호버 시 shadow-lg |
| 수치 강조 | --text-h1 크기 / --color-primary / 하단 라벨 --text-caption |

### 섹션 5: Do's & Don'ts
- Do: 섹션 간 120px 이상 여백 — 콘텐츠 호흡
- Do: CTA를 Hero + 중간 + 하단 3회 반복 — 전환율
- Do: 고객사 로고 최소 6개 — 사회적 증거
- Don't: 텍스트 블록 3줄 초과 — 가독성
- Don't: 슬라이더/캐러셀 — 정보 은닉

## UI 발췌: 서비스 소개 (UI-002)

| 항목 | 내용 |
|------|------|
| IA 페이지 | IA-P200 (서비스 소개) |
| FN 매핑 | FN-002 (서비스 상세 조회) |
| Layout | Layout-002 참조 |

**UI 요소**

| 영역 | 요소 | 상세 |
|------|------|------|
| 상단 | 페이지 Hero | 서비스명 + 한 줄 설명 + CTA |
| 본문 | 기능 카드 그리드 | 3열 / 아이콘 + 제목 + 설명 2줄 |
| 본문 | 상세 기능 블록 | 좌우 교대 (이미지 + 텍스트) |
| 본문 | 비교표 | 요금제별 기능 비교 (3열 테이블) |
| 하단 | CTA 섹션 | "시작하기" 버튼 + 보조 텍스트 |

**인터랙션**

| 트리거 | 대상 | 동작 |
|--------|------|------|
| 스크롤 | 기능 카드 | fade-in-up (순차 등장) |
| 스크롤 | 수치 강조 | 카운트업 애니메이션 |
| 클릭 | CTA 버튼 | 회원가입 페이지 이동 |
| 호버 | 기능 카드 | shadow 강화 + translateY(-4px) |

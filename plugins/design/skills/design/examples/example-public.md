## 브랜드 시트 발췌

### 섹션 0: 디자인 방향
톤: **신뢰(Trust), 접근성(Accessible), 명료(Clear), 공익(Public)** | 타겟: 전 연령 시민

### 섹션 2: 디자인 토큰

| 토큰명 | 값 | 용도 |
|--------|-----|------|
| --color-primary | #1A56DB | CTA, 링크, GNB (공공 신뢰 블루) |
| --color-secondary | #047857 | 성공, 완료 상태 (그린) |
| --color-text | #1F2937 | 본문 텍스트 (WCAG AAA 4.5:1 이상) |
| --color-text-sub | #4B5563 | 보조 텍스트 |
| --color-bg | #FFFFFF | 기본 배경 (순백) |
| --color-bg-alt | #F3F4F6 | 섹션 구분 (연회색) |
| --color-danger | #DC2626 | 경고, 긴급 공지 |
| --font-heading | Pretendard | 제목 |
| --font-body | Pretendard | 본문 (단일 서체 통일) |
| --text-h1 | 2.25rem (36px) | 페이지 타이틀 |
| --text-h2 | 1.75rem (28px) | 섹션 타이틀 |

| --radius-card | 8px | 카드 (과도한 둥글림 지양) |
| --radius-button | 6px | 버튼 |

### 섹션 3: 컴포넌트 규칙

| 컴포넌트 | 규칙 |
|---------|------|
| 버튼(Primary) | bg #1A56DB / text white / radius 6px / 48px 높이 / min-width 120px |
| 버튼(Secondary) | border 1px #1A56DB / text #1A56DB / bg white |
| 버튼(Danger) | bg #DC2626 / text white — 긴급 공지 전용 |
| 카드(서비스) | bg white / radius 8px / border 1px #E5E7EB / hover border-color #1A56DB |
| 퀵메뉴 아이콘 | 56x56 SVG / 아이콘+라벨 세로 배치 / hover bg #EFF6FF |
| 테이블 | 헤더 bg #F9FAFB / 행 hover bg #F3F4F6 / 텍스트 좌정렬 |
| 아코디언(FAQ) | 1px bottom border / chevron 180도 / 300ms ease |
| 검색 바 | height 56px / radius 28px / 돋보기 아이콘 좌측 / 자동완성 드롭다운 |

### 섹션 5: Do's & Don'ts
- Do: 본문 최소 17px — 디지털 취약계층 가독성
- Do: 클릭 영역 최소 44x44px — 터치 접근성
- Do: 색상 대비 WCAG AA(4.5:1) 이상 — 저시력 대응
- Do: 주요 동선 3클릭 이내 도달 — 정보 접근 효율
- Don't: 자동 재생 영상/음성 — 접근성 위반
- Don't: 3단 이상 드롭다운 메뉴 — 조작 난이도
- Don't: 움직이는 배경/패럴랙스 — 전정 장애 유발
- Don't: 장식용 이미지에 빈 alt 외 텍스트 — 스크린리더 혼란

## UI 발췌: 민원 신청 목록 (UI-003)

| 항목 | 내용 |
|------|------|
| IA 페이지 | IA-P201 (민원 신청 목록) |
| FN 매핑 | FN-003 (민원 목록 조회), FN-004 (민원 검색/필터) |
| Layout | Layout-002 참조 |

**UI 요소**

| 영역 | 요소 | 상세 |
|------|------|------|
| 상단 | 브레드크럼 | 홈 > 민원 > 신청 |
| 상단 | 검색 바 | 키워드 검색 + 카테고리 셀렉트 |
| 상단 | 필터 | 분야별(건축/세무/복지...) 탭 또는 칩 |
| 본문 | 민원 리스트 | 테이블형: 분류/민원명/처리기간/신청 버튼 |
| 하단 | 페이지네이션 | 숫자형 중앙 정렬, 20건/페이지 |

**상태별 화면**

| 상태 | 표시 |
|------|------|
| Default | 테이블 리스트 + 검색/필터 |
| Loading | 행 스켈레톤 (5행) |
| Empty | "검색 결과가 없습니다" + 키워드 수정 안내 |
| Error | "목록을 불러올 수 없습니다" + 새로고침 버튼 + 전화 문의 안내 |

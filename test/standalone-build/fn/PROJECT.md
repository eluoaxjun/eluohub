## 2. 배경 (BG)
| 항목 | 내용 |
|------|------|
| 프로젝트 목적 | 소규모 헬스장·PT 스튜디오 운영자를 위한 회원 관리 + 수업 예약 웹 플랫폼 구축 |
| 현행 문제점 | 엑셀·카카오톡으로 수작업 회원 관리, 예약 충돌 빈번, 매출 현황 파악 어려움 |
| 기대 효과 | 예약 자동화, 회원 이탈 감소, 매출 리포트 자동화로 운영 효율 30% 향상 |
| KPI 목표 | 런칭 3개월 내 가입 헬스장 50곳, 월 활성 회원 2,000명, 예약 전환율 70%+ |

| 항목 | 내용 |
|------|------|
| 메인 컬러 | 에너지 오렌지 (#FF6B35) |
